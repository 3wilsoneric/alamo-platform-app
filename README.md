# Alamo Platform App

Standalone Communities/Ops app extracted from the current DataHub experience.

## Purpose

- keep the DataHub shell and page experience
- add Microsoft authentication
- package toward Microsoft Store delivery
- consume Pipeline data without embedding Pipeline UI

## Start

```bash
cd /Users/eric/CareEngineMain/alamo-platform-app
npm install
npm run dev
```
